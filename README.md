# Student Management System

Developed a Student Management System with user authentication, student record management, course enrollment, attendance tracking, and grade management.

**Technologies Used**: React-JS, Spring Boot, MySQL

**Key Features**:

- User Authentication: Secure login for students, teachers, and administrators with role-based access.

- Student Record Management: Stores student details, academic history, and personal information.
- Assessment Management: Enables teachers to create, assign, and evaluate student assessments.

- Attendance Tracking: Monitors student attendance with automated reports.

- Grade Management: Enables teachers to assign and update student grades.

- Exam Management: Schedules exams, and records student performance.
